# 8917 Content Ingest

## 定位

这是一个 **工具型 / Tool Wrapper** skill。

它只负责一件事：
- **把 URL 或内容源转换成可读正文（Markdown / 文本 / transcript）**

它不负责：
- 8917 视角分析
- 飞书归档
- 后续知识沉淀

---

## 核心流程

### Step 1：识别内容来源
根据输入判断来源类型：
- X / Twitter
- 通用网页 / 新闻 / 博客
- 微信公众号
- 视频平台（抖音 / B站 / YouTube）

### Step 2：选择抓取方法
按来源类型选择最轻、最快、最稳的方法。
详细方法读取：
- `references/content-source-methods.md`

### Step 3：输出标准结果
输出内容应尽量包括：
1. 来源 URL
2. 标题（如可提取）
3. 正文 Markdown / 文本 / transcript
4. 如果抓取失败，说明失败点和下一个兜底方案

---

## 输出要求

默认输出以“可供下游分析和归档继续处理”为目标。
不要在这里加入 8917 分析，不要在这里做飞书归档。

---

## 一句话原则

**先把内容抓干净，再交给下游 skill 做分析或归档。**

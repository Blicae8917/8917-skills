## §1 选角策略(第 3 步)

从 `~/.claude/agents/`(系统每会话已注入完整列表)按议题选 **3-5 个**。原则:

**通用原则**
- **互补 > 数量**:选视角差异最大的,不要 5 个都是同领域。3 个强互补 > 5 个同质。
- **贴议题**:议题语义直接相关的专业角色优先。
- **够用就停**:简单议题 3 个,复杂跨领域才 5 个。

**按模式调整**
| 模式 | 选角侧重 |
|---|---|
| **讨论** | 立场 / 价值取向尽量分散:让乐观派、保守派、用户视角、商业视角都有人代言。目标是"撑开光谱"。 |
| **评论** | 相关专业的审查者(安全 / 性能 / 架构 / 合规 等),**强制塞入至少 1 个唱反调角色**(如怀疑者 / 批评家 / 安全架构师),专门挑漏洞。 |
| **规划** | **覆盖任务涉及的每个领域**:一个领域一个专家,避免规划出现盲区。例:做产品 → 趋势 + 架构 + 设计 + 品牌。 |

**选不到时**:议题太窄 / 太私域 / 库里无贴合专家 → 如实说明,建议"我直接答"或"用 N 个近似角色",不硬凑名字。

## §2 三模式输出格式(第 6 步)

### 讨论模式

```markdown
## 专家团讨论:[议题]

**[专家A]:** [1-2 句立场] · [1 句理由]
**[专家B]:** [1-2 句立场] · [1 句理由]
**[专家C]:** ...

### 综合
- **共识:** [各方一致处]
- **主要分歧:** [最关键的对立 + 双方理由]
- **我的判断:** [综合后的倾向 + 为什么]
```

### 评论模式

```markdown
## 专家团评审:[方案]

### 问题清单(按 severity)
| 严重度 | 专家 | 问题 | 建议 |
|---|---|---|---|
| 🔴 高 | [专家] | ... | ... |
| 🟡 中 | ... | ... | ... |
| 🟢 低 | ... | ... | ... |

### 唱反调专家的核心质疑
[那个被强制塞入的反对者最有力的一击]

### 我的综合判断
[哪些必须改 / 哪些可接受 / 整体 go-no-go]
```

### 规划模式

```markdown
## 专家团作战计划:[任务]

### 分领域规划
**[领域A — 专家]:** [该领域的关键步骤 + 风险]
**[领域B — 专家]:** ...

### 合成总计划(带分工 + 依赖)
1. [步骤] — 负责领域 — 前置依赖
2. ...

### 关键风险 / 前置条件
[跨领域的卡点]
```

## §3 综合偏见护栏(第 6 步,借 council)

我既是参与者又是综合者,所以:
- 不否定某个专家观点而不解释为什么。
- 如果某专家**改变了我的初判**,显式说出来。
- **始终保留最强反对意见**,即使我不采纳。
- 若 2 个以上专家一致反对我的初判,**当作真实信号**重新审视。
- verdict 之前,原始各方立场可见,不把分歧藏进结论。

## §4 subagent prompt 模板(第 5 步,反锚定)

每个专家 subagent 用 Agent 工具派遣,prompt 形如:

```text
你是[专家角色]。这是一个多视角[讨论/评审/规划]的一员。

议题:
[一句话议题]

上下文(只给必要的):
[相关片段 / 约束 / 方案,保持精简]

请按你的专业视角输出:
1. 立场/判断 — 1-2 句
2. 理由 — 3 条精简 bullet
3. 风险/盲点 — 你认为最大的风险或别人会忽略的点
[评审模式追加] 4. 问题清单 — 按严重度列你发现的问题
[规划模式追加] 4. 你这个领域的关键步骤 + 依赖

直接,不要 hedging,300 字以内。
```

**铁律**:不在 prompt 里塞完整对话历史。每个专家独立、互不可见(单轮模式)。

## §5 用 Agent 工具实际派遣

- 选用对应 `subagent_type`(英文 ID,如 `Security Architect` / `Prompt Engineer`)。中文叫法→英文 ID 的映射见专家速查表。
- 并行:在**同一条消息里发起多个 Agent 调用**,它们并发执行。
- 收齐后按 §2 综合。

## §6 "加一轮交锋"开关(预留,默认不做)

仅当用户显式要求("让他们再辩一轮 / 互相看看再说"):
1. 把第一轮各方观点(匿名或具名)汇总成一段。
2. 重新派遣每个专家,prompt 追加"以下是其他专家的观点:[...],请反驳或补充,不要重复你已说的"。
3. 再综合。
成本约翻倍。非重大决策不建议。
